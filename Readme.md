# Asteroids

A remake of a classic, Asteroids. <br />
Built with ExcaliburJS; game art from Open Game Art.

## Compiling the game

You will need Node.js, Bower, and NPM installed.

To compile:

    npm install
    bower install
    npm start

## Controls

1. Move with A,W,S,D
1. Aim with arrow keys
1. Shoot with Spacebar
